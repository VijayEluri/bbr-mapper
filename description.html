Autonomous 2-D Mapping of a Building Floor

by Constantin Berzan & Andy Sayler

Objective:

We aim to program a robot that purposefully navigates a building and uses laser and vision data to create a 2D map of the floor. This involves exploring all open space, and working around obstacles. Additionally, we aim for our robot to be able to localize itself within the mapped building, i.e. figure out where it is based on its surroundings.

<short project summary including how this project uses behavior-based techniques>

Final demonstration:

The demo will happen in Halligan Hall, first floor.
  • We will place a brain-washed robot somewhere on the floor, and it will start exploring. It will navigate all available open areas without bumping into obstacles. When done, it will produce an image representation of the map it has constructed.
  • After the map has been created, we will turn off the robot and move it to a specific room. We will then turn it on and let it figure out where it is. It will draw its inferred location on the image representation of the map.

Robot platform:

Videre platform equipped with lasers and camera.
